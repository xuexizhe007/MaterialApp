物资管理系统 V1.3 (Flutter App版)

【如何运行】
1. 确保电脑已安装 Flutter SDK (https://flutter.dev)。
2. 将本文件夹所有内容解压。
3. 在终端进入该文件夹，运行命令 `flutter pub get` 下载依赖。
4. 连接手机或启动模拟器。
5. 运行 `flutter run`。

【功能说明】
1. 登录：输入任意用户名登录，App会记住当前操作员。
2. 目录与入库：
   - 目录页可以新增物资。
   - 入库页点击 "模拟扫码" 可随机匹配已有物资或提示不存在。
3. 出库：
   - 自动校验库存，库存不足无法提交。
   - 需填写领用部门与人名。
4. 数据：
   - 所有数据保存在手机本地 (Shared Preferences)。
   - 重启 App 数据不丢失。
5. PDF 导出：
   - 在 "设置/库存" -> "出入库记录" 中，点击打印图标即可调用系统打印服务生成 PDF。
   - 注意：由于演示代码未包含大体积中文字体包，PDF中的中文可能显示为方框，这是正常现象，需在生产环境中引入 TTF 字体文件解决。

【技术栈】
- Flutter (Dart)
- Provider (状态管理)
- SharedPreferences (本地存储)
- Pdf/Printing (单据生成)
